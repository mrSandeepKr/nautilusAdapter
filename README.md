# Nautilus Project

This project contains a Python environment setup for algorithmic trading using [Nautilus Trader](https://github.com/nautechsystems/nautilus_trader).

## Setup

1. Run the setup script to create a virtual environment and install dependencies:
   ```bash
   ./setup_venv.sh
   ```
2. Activate the virtual environment (or use `direnv` which will automatically source `.envrc`):
   ```bash
   source venv/bin/activate
   ```
